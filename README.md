# Simple NodeJS-RestAPI Boilerplate with ORM, API Docs, Docker
#### Simple Rest API with nodejs, express, sequelize (orm), swagger (API Docs), Docker

## Feature
- Auto Migrate (Sequelize)
- API Docs (Swagger)
- Unit Test (JestJS)
- Docker Config

## Dependencies
- dotenv
- cors
- express
- mysql2
- sequelize
- swagger-jsdoc
- swagger-ui-express
- jest (Dev)
- nodemon (Dev)
